## Installation

### Requirements
* [Node 16.3+](https://nodejs.org/en/)

### Installation

``` bash
# Install dependencies
npm install @vonage/js-math-utils
```

### Running tests

``` bash
# Running the tests
npm run test

# Running the test coverage
npm run coverage
```
